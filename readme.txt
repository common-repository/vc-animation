== Description ==

VC Animation is a Visual Composer add-on helps you add animation for any elements. Impress your visitors with many effects, eg. floating, shake, heartbeat, spin...

= Live demo =

Click to see [live demo](http://demo.wpclever.net/?item=vcla "live demo")

= Features =

- 20+ pre-defined effects
- Easy to change the duration
- Easy to change the delay
- Easy to change other animation properties

= Need support? =

Visit [plugin documentation website](https://wpclever.net "plugin documentation")

== Installation ==

1. Please make sure that you installed Visual Composer plugin
2. Unzip the vc-animation.zip
3. Copy vc-animation folder to wp-content/plugins
4. Go to Plugins / Installed Plugins, find VC Animation and click Activate
5. Now when add/edit the page with Visual Composer you can add VC Animation element
6. Setting for this animation as you want, you also can see the preview
7. Add other elements inside VC Animation
8. Save this page and see the result
